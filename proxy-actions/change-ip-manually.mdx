Changing your proxy's IP is extremely easy with Illusory. There are multiple ways to perform this action.

### Methods to change proxy IPs

- Proxy Modal
- Proxy Card
- API

### Method 1: Change Proxy IP via Proxy Modal

Open the proxy modal and click the "Change IP" action button
