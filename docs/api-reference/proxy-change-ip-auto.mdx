---
title: 'Auto Change IP'
openapi: 'POST /v1/proxies/autochangeip/{proxyName}'
---

Changes the auto-change settings for of a proxy. Auto-change controls whether a proxy automatically changes its external IP on a regular schedule.


<Info>
    You can set a proxy's change `rate` (the rate at which it automatically changes IPs) for every `1` to `1440` minutes.`
</Info>
