### Change IP methods

Changing your proxy's IP is extremely easy with Illusory. There are multiple ways to perform this action.

- Proxy Modal
- Proxy Card
- Automatically
- API Link

<Warning>You may only reset your proxy's IP every 1 minute.</Warning>
